## Generate and build

To build this project, open "Terminal", go to your project folder and type following lines:

``` cmake
mkdir build
cd build
cmake .. -G "Xcode"
open ./HelloWorld.xcodeproj
```

Select `HelloWorld` project and type Cmd+R to build and run it.
